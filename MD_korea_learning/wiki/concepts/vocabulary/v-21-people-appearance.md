## 🕸️ 3. Deep Semantic Network — Mạng Ngữ Nghĩa Sâu
> Học từ theo cụm và mạng lưới để não bộ tự động liên kết khi giao tiếp.

### 🕸️ Từ tâm: **멋있다** (**Ngầu / Đẹp / Phong cách**)
- 🔄 **Synonyms (Từ liên quan):** 잘생기다 (Đẹp trai - chỉ dùng cho nam), 예쁘다 (Đẹp gái/Đẹp).
- ⚖️ **Antonyms (Trái nghĩa):** 못생기다 (Xấu xí).
- 🧩 **Word Family (Cụm đi kèm):** 얼굴이 잘생기다 (Mặt đẹp trai), 옷을 멋있게 입다 (Mặc quần áo ngầu).

## 🎯 5. Mnemonic Lab — Mẹo Ghi nhớ Siêu tốc (**Keyword Method**)
> Liên tưởng âm thanh Hàn → Việt để ghi nhớ nhanh.

| Từ | Phiên âm | Mẹo ghi nhớ |
| :--- | :--- | :--- |
| 키 | khi | **KHI** = "kì" → **chiều cao** = kì (**so kè ai cao**) |
| 날씬하다 | nal-ssin-ha-da | **NAL SSIN** = "nải xỉn" → **thon thả** = nải chuối xỉn nhỏ |
| 뚱뚱하다 | ttung-ttung-ha-da | **TTUNG** = "tủng" → **mập** = tủng tủng tròn ú |
| 잘생기다 | jal-xaeng-gi-da | **JAL XAENG** = "giỏi sinh" → **đẹp trai** = sinh ra giỏi! |
| 예쁘다 | ye-ppư-da | **YE PPƯ** = "yeah đẹp!" → **đẹp** = "Yeah đẹp!" |
| 어리다 | ơ-ri-da | **Ơ RI** = "ơ rì" → **trẻ** = "ơ" sao trẻ rì vậy? |
| 늙다 | nưk-da | **NƯK** = "nức" → **già** = nức nở vì già |
| 머리카락 | mơ-ri-kha-rak | **MƠ RI** = "mờ rì" → **tóc** = sợi mờ rì rì |

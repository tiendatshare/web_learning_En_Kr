- 🧩 **Word Family (Cụm đi kèm):** 여행사 (Công ty du lịch), 여행지 (Địa điểm du lịch).

## 🎯 5. Mnemonic Lab — Mẹo Ghi nhớ Siêu tốc (Keyword Method)
> Liên tưởng âm thanh Hàn → Việt để ghi nhớ nhanh.

| Từ | Phiên âm | Mẹo ghi nhớ |
| :--- | :--- | :--- |
| 여행 | yơ-haeng | **YƠ HAENG** = "du hành" → **du lịch** = du hành (Hán Việt!) |
| 공항 | gong-hang | **GONG HANG** = "không hàng" → **sân bay** = không cảng (Hán Việt!) |
| 호텔 | ho-thel | **HOTEL** → từ mượn quốc tế! |
| 경치 | gyơng-chi | **GYƠNG CHI** = "cảnh trí" → **phong cảnh** = cảnh trí (Hán Việt!) |
| 관광 | gwan-gwang | **GWAN GWANG** = "quan quang" → **tham quan** (Hán Việt!) |
| 여권 | yơ-gwơn | **YƠ GWƠN** = "dư quyền" → **hộ chiếu** = quyền đi nước ngoài |
| 예약하다 | ye-yak-ha-da | **YE YAK** = "dự ước" → **đặt trước** = dự ước (Hán Việt!) |
| 사진 | xa-jin | **XA JIN** = "xạ chân" → **ảnh** = tả chân (Hán Việt!) |

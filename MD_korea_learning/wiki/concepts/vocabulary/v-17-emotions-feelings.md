## 🕸️ 3. Deep Semantic Network — Mạng Ngữ Nghĩa Sâu
> Học từ theo cụm và mạng lưới để não bộ tự động liên kết khi giao tiếp.

### 🕸️ Từ tâm: **기분이 좋다** (**Tâm trạng tốt**)
- 🔄 **Synonyms (Từ liên quan):** 기쁘다 (Vui vẻ), 즐겁다 (Vui thú), 행복하다 (Hạnh phúc).
- ⚖️ **Antonyms (Trái nghĩa):** 우울하다 (U uất/Buồn chán), 기분이 나쁘다 (Tâm trạng tồi tệ).
- 🧩 **Word Family (Cụm đi kèm):** 눈물을 흘리다 (Chảy nước mắt), 웃다 (Cười).

## 🎯 5. Mnemonic Lab — Mẹo Ghi nhớ Siêu tốc (**Keyword Method**)
> Liên tưởng âm thanh Hàn → Việt để ghi nhớ nhanh.

| Từ | Phiên âm | Mẹo ghi nhớ |
| :--- | :--- | :--- |
| 기쁘다 | gi-ppư-da | **GI PPƯ** = "ghì phừ" → **vui** ghì chặt lấy mà cười phừ |
| 슬프다 | xưl-phư-da | **XƯL PHƯ** = "xưa phũ" → **buồn** vì xưa bị phũ |
| 화나다 | hwa-na-da | **HWA NA** = "hỏa nạ" → **tức giận** = hỏa nạ (**lửa nổ**) |
| 걱정하다 | gơk-jơng-ha-da | **GƠK JƠNG** = "cực trọng" → **lo lắng** vì chuyện **cực** trọng |
| 행복하다 | haeng-bok-ha-da | **HAENG BOK** = "hạnh phúc" → **hạnh phúc** (**Hán Việt!**) |
| 외롭다 | we-rop-da | **WE ROP** = "ối rồi" → **cô đơn** = "ối rồi" một mình |
| 피곤하다 | phi-gon-ha-da | **PHI GON** = "phì gòn" → **mệt** = phì phì ngáp |
| 신나다 | xin-na-da | **XIN NA** = "xỉn nà" → **hào hứng** = xỉn (**say**) vui nà! |
